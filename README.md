# fastapi temp

This is a simple FastAPI(backend framework) app called as "Cost-Manager" which I built as part of my practice.

🧩 Main features & tools:
- Multiple language support(Persian & English)
- Serializers & Pydantic models
- SQLAlchemy ORM + Alembic migrations
- JWT (cookie-based authentication)
- Redis cache
- PostgreSQL database
- Docker & Docker Compose
- CI/CD pipeline(using github Actions and hamravesh)
- Testing with Pytest
- Code quality via Flake8 & Black
- Load testing with Locust
- Error tracking with Sentry
